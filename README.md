# HR Analytics Dashboard

## Project Overview
This project presents an interactive HR Analytics Dashboard built using Power BI to analyze employee attrition patterns and workforce demographics. The dashboard enables HR stakeholders to identify key drivers of attrition and support data-driven decision making.

---

## Objectives
- Analyze employee attrition trends
- Identify high-risk demographic groups
- Understand role-based and satisfaction-based attrition
- Provide actionable workforce insights

---

## Tools & Technologies
- Power BI
- Data Visualization
- Data Cleaning & Transformation
- HR Analytics Dataset

---

## Dashboard Features
- KPI Cards (Total Employees, Attrition Count, Avg Age, Salary)
- Attrition by Education
- Attrition by Age Group
- Attrition by Gender
- Attrition by Job Role
- Job Satisfaction Analysis
- Experience Before Attrition Trends
